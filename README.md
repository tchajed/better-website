# Improving your website

[Demo](https://tchajed.github.io/better-website/)

If your website has no styling, please add this to your `<head>` element:

```html
<style>
  body {
    margin: 2em auto;
    max-width: 800px;
    font-size: 18px;
    padding: 1em;
    text-align: justify;
  }
  h1,
  h2,
  h3 {
    line-height: 1.2;
  }
</style>
```
